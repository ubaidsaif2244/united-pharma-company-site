## Image Naming Convention

All images should follow this naming pattern:
- `product-name-strength.jpg` (e.g., `amoxicillin-500mg.jpg`)
- Use lowercase letters and hyphens for spaces
- Include strength/dosage in the filename
- Use `.jpg` format for consistency

## Image Specifications

- **Format**: JPG/JPEG
- **Size**: 400x400 pixels (square format)
- **Quality**: High quality, clear product shots
- **Background**: White or transparent background preferred
- **File Size**: Keep under 200KB for optimal loading

## Fallback Images

If real images are not available, the system will fall back to generic SVG icons:
- `/images/products/tablet-generic.svg`
- `/images/products/capsule-generic.svg`
- `/images/products/syrup-generic.svg`
- `/images/products/injection-generic.svg`

## Upload Instructions

1. Take clear, high-quality photos of each product
2. Crop to square format (1:1 aspect ratio)
3. Resize to 400x400 pixels
4. Save as JPG with appropriate filename
5. Upload to this directory
