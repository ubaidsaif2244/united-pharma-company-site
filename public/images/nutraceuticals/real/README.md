# Real Nutraceutical Product Images Directory

This directory contains real nutraceutical product images for the United Pharma website.

## Image Naming Convention

### Nutraceutical Products:
- `vitamin-d3-real.jpg` - Vitamin D3 supplement bottles/packaging
- `omega3-real.jpg` - Omega-3 fish oil bottles/capsules
- `turmeric-real.jpg` - Turmeric curcumin bottles/capsules
- `immune-complex-real.jpg` - Immune defense supplement packaging
- `collagen-real.jpg` - Collagen beauty blend containers
- `b-complex-real.jpg` - B-Complex vitamin bottles
- `probiotics-real.jpg` - Probiotics supplement containers
- `ashwagandha-real.jpg` - Ashwagandha herbal supplement bottles
- `energy-boost-real.jpg` - Energy boost formula packaging

### Image Specifications:
- **Format**: JPG, PNG, or WebP
- **Dimensions**: 400x400px (1:1 aspect ratio for product cards)
- **File Size**: Under 500KB for optimal loading
- **Quality**: High resolution, professional product photography

### Usage:
These images will be automatically used in the Nutraceuticals Products page when available. The system will fallback to generic supplement illustrations if real images are not found.

### Upload Instructions:
1. Place your real nutraceutical product images in this directory
2. Follow the naming convention above
3. Ensure images meet the specifications
4. The website will automatically detect and use these images
